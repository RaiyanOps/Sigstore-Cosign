After complete the installation proceed with the below steps

Go to the path where you generated the key pair like private and public:

its has a file called : cosign.key ----> sign the image using private key
                        cosign.pub ----> verify the image using the public key

sample generation output:
-------------------------

root@rs1-vm-deem-dev:~/cosign/new-setup# cosign generate-key-pair
Enter password for private key:
Enter password for private key again:
Private key written to cosign.key
Public key written to cosign.pub

root@rs1-vm-deem-dev:~/cosign/new-setup# ls
cosign.key  cosign.pub 




helm repo add kyverno https://kyverno.github.io/kyverno/
helm repo update
helm install kyverno kyverno/kyverno -n cosign --create-namespace


after installing kyverno helm create a policy	

kubectl create secret generic cosign-pub --from-file=cosign.pub -n cosign

after deployin the kyverno helm command you need to apply the policy 

please check the deployment folder
